button_green_up.png buat 5 buah animasikan ke green down jika ditekan
tombol idnya :
1. tombol purge jika ditekan akan mengaktifkan animasi fan fan akan hidup selama 8 detik kemudian lampu purge completed akan menyala menandakan purge selesai
2. tombol LT jika ditekan akan menghidupkan semua lampu, untuk mengecek kondisi lampu indikator
3. Tombol pilot, jika ditekan maka, valve blowdown pilot akan menutup, valve pilot upstream terbuka disusul dengan valve downstream membuka, kemudian api pilot hidup. tombol ini hanya bisa mengaktifkan valve sekali pencet, jika dipencet lagi tidak akan terjadi apa-apa, kecuali tombol reset dipencet dan semua kondisi akan Kembali pada posisi awal. (beri jeda 2 detik untuk reset)
4. Tombol main burner, jika ditekan maka akan menutup valve blowdown main burner dan membuka valve upstream main burner disusul dengan valve downstream main burner. tombol ini hanya bisa mengaktifkan valve sekali pencet, jika dipencet lagi tidak akan terjadi apa-apa, kecuali tombol reset dipencet dan semua kondisi akan Kembali pada posisi awal. (beri jeda 2 detik untuk reset)
5. Tombol reset, tombol ini untuk mereset semua kondisi dan alarm, jika tombol ini ditekan maka semua animasi akan direset keposisi awal lagi. dan sequence harus dilakukan diawal lagi
